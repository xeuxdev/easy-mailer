# Introduction

Welcome to the Easy Mailer Documentation

## What is EasyMailer?

**Easy Mailer** is a helper function that simplifies the process of sending emails in your` Node.js` apps with `little configuration`, using nodemailer.

Embracing the ethos of open source, we strive to contribute to the community by making email sending effortless for developers working on side projects, all while ensuring deliverability and analytics. Free for all.
